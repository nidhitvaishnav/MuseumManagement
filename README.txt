# MuseumManagement
This PHP project was created using wamp server on the museum management system as a part of PHP learning.  
It was created in 2nd year of Undergraduate study in 2015.
It has basic database.

This project is intended for the museum employees.
Here, employee can register and login with their user id and password.
They can 
- see the stored museum archives based on the department, 
- add the artifacts, 
- see the museum blueprints,
- see the employee details,
- add and see publications by museum,
- search employees
- search artifacts using its name or the name of artist
